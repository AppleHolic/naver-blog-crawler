# Naver Blog Crawler

## Requirements

- Python 2.7+
- `pip install beautifulsoup4`

## Run

    $ python crawler.py
    What kind of blog is it?    # Insert the kinds of blog

### List of available blog types (kinds)

- food
- tour
- technology
- ...
